    • Parscore Excel Spreadsheet Conversion Tool

      With the adoption of the Canvas Learning Managment System, faculty relying on ITS support faced the challenge that the new Canvas system did not have a compatible tool to convert their grading spreadsheets into CSV files for the Scantron ParScore grading program to use.       Git-Hub Project Link

    • Audio to Subtitle Amazon Web Services Project

      In the Fall and Winter of 2019, we were asked to look into using Amazon Web Services to build a tool for converting lecture audio into subtitles for use in BlackBoard and Canvas. Our lecture video and audio were recorded using the MediaSite lecture capture software. By leveraging the MediaSite REST API we would retrieve the audio files necessary and then upload them to our AWS storage. Then, using the AWS Boto3 Python API, a series of scripts would send job requests to the AWS client and upload files into the AWS Machine Learning Audio Transcription suite. This suite returns a JSON file of the detected speech-to-text content with time stamps and confidence values. This file would then be converted to the standard .srt subtitle format and be used alongside lecture video.
